## Home AI CCTV Web Application  
<br>

### 기능1. AI가 감시중인 실시간 CCTV 영상을 확인 할 수 있고
<br>

### 기능2. 사람이 항상 CCTV를 감시하기 어렵기 때문에 AI가 대신 CCTV 영상을 감시하다가 침입을 감지하면 사진을 저장하고 알림 메시지를 보냅니다  
<br>

### 기능3. 침입을 감지하는 기준, 예를 들어 사람과 자동차만 인식하도록 하거나 AI모델 종류와 사진 저장 주기 변경 등 설정값을 입력받아 AI CCTV 영상 감시를 할 수 있습니다  
<br>

### 기술 study. 인프라 구축부터 운영 배포까지 활용한 기술들의 study 기록과 (01~11) 실제 AI CCTV 실행 소스코드(99) 를 확인 할 수 있습니다
<br>
<br>
<br>
