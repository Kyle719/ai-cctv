■■■ 12강 DNS record & CNAME
DNS 에는 record type 이 여러가지 있다.
A type DNS record 는 도메인과 ip 주소를 매핑
ex) example.com 을 192.0.1.1 로 연결
CNAME type DNS record 은 도메인에 대한 별명을 지어주는것. 도메인을 도메인에 매핑
ex) www.example.com 으로 들어오면 example.com 으로 매핑.
www.example2.com 으로 들어오면 example.com 으로 매핑.
이렇게 여러가지 별명 붙일수있다.
example.com 의 ip 주소가 자주 바뀔때 유용하다.
www.example.com 의 ip, www.example2.com 의 ip 안 바꿔줘도 된다.
